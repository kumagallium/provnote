# プライバシーポリシー

**最終更新日:** 2026-04-27

Graphium はオープンソースのローカルファーストノートエディタです。このポリシーでは、アプリケーションがデータをどのように扱うかを説明します。

## Graphium について

Graphium は、プロヴェナンス（来歴）追跡機能付きのブロックベースノートエディタです。ノートはあなたのファイルシステム（デスクトップ版）またはブラウザの IndexedDB（Web 版）に保存されます。すべての編集はローカルで行われ、Graphium 自身がノートをどこかに送信することはありません。

## データ収集

**Graphium は個人データを一切収集・保存・送信しません。**

- アクセス解析やトラッキングなし
- Cookie なし
- アカウント・サインイン不要
- サーバー側でのノート保存なし

## データの保存先

| データ | 保存先 |
|--------|--------|
| ノート（デスクトップ版） | あなたのファイルシステム上のプレーン JSON ファイル（既定: `~/Documents/Graphium/`、設定で変更可能） |
| ノート（Web / Docker セルフホスト） | ブラウザの IndexedDB |
| 個人情報 | どこにも保存しない |

デスクトップ版の保存先を Google Drive / iCloud / Dropbox などのクラウド同期フォルダに指定すると、OS のクラウド同期によってファイルがそのプロバイダに複製されます。Graphium 自身がクラウドストレージ API と通信することはありません。

## サードパーティサービス

Graphium はストレージや認証に関してサードパーティサービスを利用していません。

AI 機能（オプション、オプトイン）を有効にするために LLM プロバイダを設定した場合、プロンプトはそのプロバイダの規約に従って送信されます。Graphium はこの通信を中継・記録しません。

## 子どものプライバシー

Graphium は 13 歳未満の子どもから意図的に情報を収集することはありません。
